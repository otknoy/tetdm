
TETDMの起動方法には以下の3つがあります。

1)　アイコンをダブルクリックする（入力ファイルを後で読み込む）
2)　アイコン上に入力ファイルをドラッグ＆ドロップする
3)　ターミナルまたはコマンドプロンプトを利用する(プログラマ向け)

page
1) アイコンをダブルクリックする（入力ファイルを後で読み込む）

　　TETDMフォルダ内の、
　　「TETDM.jar」か「TETDM512.jar」をダブルクリックします。

　　TETDM512.jarは、512MBのメモリを確保して起動します。
　　大きめのテキストファイルを扱う場合は、こちらを選択します。

page
2)　アイコン上に入力ファイルをドラッグ＆ドロップする

　　用意した入力テキストファイルを、
　　TETDMフォルダ内の、下記ファイルの上にドラッグ＆ドロップします。
　　Windowsの場合：「TETDM.bat」
　　Macの場合：「TETDM.app」

　　TETDM.batは、1024MB
　　TETDM.appは、512MBのメモリを確保します。

3)　ターミナルまたはコマンドプロンプトを利用する(プログラマ向け)

　　コマンドを入力できるソフト（例えば下記）を起動します。
　　Windowsの場合：「コマンドプロンプト」
　　Macの場合：「ターミナル」

　　TETDMフォルダ内に、cdコマンドで移動します。

　　以下のコマンドでTETDMを起動できます。

　　入力ファイルがない場合：
　　java -jar TETDM.jar
　　入力ファイルがある場合（textフォルダ内のファイル）：
　　java -jar TETDM.jar text/urashima.txt
　　確保するメモリの量を指定する場合：
　　java -Xmx512m -jar TETDM.jar text/urashima.txt
