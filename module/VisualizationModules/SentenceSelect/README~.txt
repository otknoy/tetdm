フォーカス文を選択できるチェックボックスを表示
文選択(SegmentSelect(ID=1101))

チェックボックスにより，フォーカスをあてる文の集合を明示的に選択できます．
チェックボックスがチェックされると，
選択された文がフォーカス文として情報が上書きされます．

public void setData(int dataID, double data[])

public void setData(int dataID, String data)
